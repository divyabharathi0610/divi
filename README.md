# 🌐 Modern Responsive Website

A modern, responsive website built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates a clean user interface with smooth navigation, dark mode support, animations, and a contact form.

---

## 📖 Overview

This website is designed as a simple landing page that can be used for personal portfolios, business websites, or learning front-end web development.

The project focuses on responsive design, user-friendly navigation, and interactive features using only HTML, CSS, and JavaScript.

---

## ✨ Features

- 📱 Fully Responsive Design
- 🎨 Modern User Interface
- 🌙 Dark/Light Mode Toggle
- 🧭 Sticky Navigation Bar
- 🔗 Smooth Scrolling Navigation
- 💼 Services Section
- ℹ️ About Section
- 📩 Contact Form
- 🎬 Scroll Animations
- ⚡ Lightweight and Fast

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/modern-website.git
```

### 2. Open the Project

Navigate to the project folder.

```bash
cd modern-website
```

### 3. Run the Website

Simply open **index.html** in your preferred web browser.

No installation or additional dependencies are required.
